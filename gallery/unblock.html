Unblock

About:

Wanting to create a web application that provided inspiration (and inspired by a friend of mine, Amanda Lugo), I created UnBlock. Using the a word generator, I randomly pull words that are a certain part of speech determined by the user. The end result is a randomly generated word from which they can draw inspiration for their art form of choice.

Tool(s) used:

HTML/CSS for styling, Wordnik's API for data generation, jQuery for styling and AJAX calls, Colourlovers for color inspiration.
